# [Advent Of Code 2023](https://adventofcode.com/2023)


Bun installation:  
https://bun.sh/docs/installation  

Rust installation:  
https://rustup.rs/

To install dependencies:

```bash
bun install
```

To list all available commands:

```bash
bun run
```

To run:

```bash
bun day14
bun day14-test
bun day14-test-rust
```
